# typescript

## 安装 typescript

```bash
yarn add --dev typescript
```

## 初始化 typescript 配置

```bash
./node_modules/.bin/tec --version  // 查看版本号
./node_modules/.bin/tsc --init // 生成 tsconfig.json 文件
```

### 编译 ts 文件为 js文件

```bash
./node_modules/.bin/tsc xxx.js xxx.ts
```

### ts 配置 jQuery

```bash
yarn add --dev @types/jquery  // 添加jQuery类型
```
